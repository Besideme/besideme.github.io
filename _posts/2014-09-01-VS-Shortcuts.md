- 格式化代码 `Ctrl + K Ctrl + D`

- 复制/剪切/删除整行，光标移至该行， `Ctrl + C`  `Ctrl + X`  `Ctrl + L`  

- 框式选择 `Ctrl + Alt + 上下键`  

- 选中从光标起到行首/行尾 `Shift + Home`  `Shift + End`  

- 注释/取消注释 `Ctrl + K + C`  `Ctrl + K + U`  

- 查找/替换 `Ctrl + F`  `Ctrl + H`  

- 转小写/转大写 `Ctrl + U`  `Ctrl + Shift + U`

- 注释<%--  --%> `Ctrl + K Ctrl + C` 

- 全部保存 `Ctrl + Shift + S`
